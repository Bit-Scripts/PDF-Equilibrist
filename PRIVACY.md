# Privacy Policy — PDF-Equilibrist

PDF-Equilibrist is a local desktop application. It does not collect,
transmit, or store any personal data.

- No telemetry
- No analytics
- No network connection during normal use
- All PDF processing is performed locally on your device

The only optional network access is:
- Checking for updates (GitHub Releases API, on user request)
- CVE scanning of installed packages (OSV.dev API, on user request)

No personal data is sent in either case.
