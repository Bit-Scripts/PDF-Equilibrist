## Génération des PDFs

```powershell
python tools/build_docs_pdf.py
# → docs/pdf/1.INDEX.pdf  (avec liens cliquables entre documents)
# → docs/pdf/2.ARCHITECTURE.pdf
# → docs/pdf/3.CONTRIBUTING.pdf
# → docs/pdf/4.OPERATIONS.pdf
# → docs/pdf/5.BUILD.pdf
```
